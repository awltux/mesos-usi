AsyncCuratorBuilderSettings 

case class AsyncCuratorBuilderSettings(createOptions: Set[CreateOption] = ..., deleteOptions: Set[DeleteOption] = ..., existsOptions: Set[ExistsOption] = Set.empty, createMode: CreateMode = CreateMode.PERSISTENT, acl: Seq[ACL] = Seq.empty, nodeVersion: Int = -1, compressedData: Boolean = true) extends Product with Serializable

Simple case class holding factory's configuration options. Sensible defaults are provided for all options.

createOptions

a set with create options. By default node parents are created if needed and node data is compressed

deleteOptions

a set with delete options. By default node children are deleted if the node doesn't exist the delete method will appear to succeed

existsOptions

a set with exists options. By default no options are set

createMode

node create mode. By default CreateMode.PERSISTENT nodes are created

acl

node ACL. By default an empty list is used

nodeVersion

node version used to update node data or delete it. By default, -1 is used which matches all versions

compressedData

true if node data compression is enabled. Compression is enabled by default
