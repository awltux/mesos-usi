CoreHelloWorldFramework 

Companion object CoreHelloWorldFramework

case class CoreHelloWorldFramework(frameworkId: FrameworkID, killSwitch: KillSwitch, result: Future[Done]) extends Product with Serializable

Run the hello-world example framework that:

  • relies on our core module to handle Mesos connection, event handling, offer matching etc. as opposed to the simple-hello-world framework which uses only the Mesos client and has to implement all of the above mentioned pieces itself
  • starts one echo "Hello, world" && sleep N task
  • exits when the task fails (or finishes)

Good to test against local Mesos.
