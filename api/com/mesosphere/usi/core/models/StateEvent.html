StateEvent 

sealed trait StateEvent extends AnyRef

Sealed trait including all events that can describe the evolution of SchedulerState (statuses and records).

These events are emitted by the SchedulerLogic in response to evolution of specification states and Mesos events. They are provided so the framework-implementation can replicate and react to the evolution of that state as best suits the framework's domain.
