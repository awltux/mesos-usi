  2.  case class FetchUri(uri: URI, extract: Boolean = true, executable: Boolean = false, cache: Boolean = false, outputFile: Option[String] = None) extends Product with Serializable

    Specification used by the [Mesos fetcher](http://mesos.apache.org/documentation/latest/fetcher/) to download resources into the sandbox directory of a task in preparation of running the task.

    Specification used by the [Mesos fetcher](http://mesos.apache.org/documentation/latest/fetcher/) to download resources into the sandbox directory of a task in preparation of running the task. Defaults taken from Mesos proto files.

    uri

    artifact URI to download

    extract

    if true, the artifact will be treated as an archive an extracted to the sandbox. See the list of the supported archive types below

    executable

    if true, the fetch result will be changed to be executable (by "chmod") for every user

    cache

    if true, the fetcher cache is to be used for the URI

    outputFile

    if set, the fetcher will use that name for the copy stored in the sandbox directory
