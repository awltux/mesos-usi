DcosServiceAccountProvider 

Companion object DcosServiceAccountProvider

case class DcosServiceAccountProvider(uid: String, privateKey: String, root: URL)(implicit system: ActorSystem, materializer: ActorMaterializer, context: ExecutionContext) extends CredentialsProvider with PlayJsonSupport with StrictLogging with Product with Serializable

Provides DC/OS session tokens as JWT for service accounts.

uid

The user id.

privateKey

The JWT secret.

root

The DC/OS root URL.

system

The ActorSystem used for HTTP requests.

materializer

The ActorMaterializer use to unmarshal HTTP responses.

context

The execution context for transforming the future HTTP response.

See also

See https://docs.mesosphere.com/1.11/security/ent/iam-api/#obtaining-an-authentication-token for details on the protocol.
