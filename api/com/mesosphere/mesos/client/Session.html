Session 

Companion object Session

case class Session(url: URL, streamId: String, authorization: Option[CredentialsProvider] = None)(implicit askTimout: Timeout) extends Product with Serializable

A session captures the connection information with Mesos. This is among other things the URL and credentials providers.

It behaves similar to Python Request's Session object. Thus it provides methods to construct and connect to Mesos.

url

The Mesos master URL.

streamId

The Mesos stream ID. See the docs for details.

authorization

A CredentialsProvider if the connection is secured.
