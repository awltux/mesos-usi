    A minimal version of a task's specification. Contains the command and the necessary resources. Note that we only consider memory and cpus for now, ignoring disk, gpus, ports etc.

    task's name

    cmd

    command to execute

    amount of CPU resources

  5.  case class Task(spec: Spec, taskId: String, state: Option[TaskState], agentId: Option[AgentID]) extends Product with Serializable

    A minimalistic Mesos task representation

    A minimalistic Mesos task representation

    spec

    task spec

    taskId

    task Id that is unique for every Mesos task. It is of the form Spec.name.UUID e.g. sleep.5e5601ab-01c7-4299-9eb1-0741763d2749

    state

    current TaskState as reported by Mesos. It's optional since it does not exists until the first status update arrives from Mesos

    agentId

    tasks agent Id. It's optional since it does not exists until the task is scheduled to run
