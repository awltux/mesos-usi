MesosCluster 

case class MesosCluster(suiteName: String, masterUrl: String, autoStart: Boolean = false, config: MesosConfig = MesosConfig(), agentsConfig: MesosAgentConfig = MesosAgentConfig(), waitForMesosTimeout: FiniteDuration = 5.minutes)(implicit system: ActorSystem, mat: Materializer) extends AutoCloseable with Eventually with StrictLogging with Product with Serializable

Basic class that wraps starting/stopping a Mesos cluster with passed agent/master configuration and configurable number of masters/agents.

suiteName

suite name that uses the cluster. will be included into logging messages

masterUrl

Mesos master ZK url in the form of zk://zk-1/mesos

autoStart

if true, the cluster will be started automatically

config

Mesos master configuration

agentsConfig

Mesos agent configuration

waitForMesosTimeout

time to wait for Mesos master/agent to start
