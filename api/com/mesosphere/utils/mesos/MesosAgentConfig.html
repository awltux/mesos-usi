MesosAgentConfig 

case class MesosAgentConfig(launcher: String = "posix", containerizers: String = "mesos", isolation: Option[String] = None, imageProviders: Option[String] = None, seccompConfigDir: Option[String] = None, seccompProfileName: Option[String] = None, executorRegistrationTimeout: Duration = 1.minute, fetcherStallTimeout: Duration = 1.minute) extends Product with Serializable

Mesos gent configuration parameters. For more information see [Mesos documentation](http://mesos.apache.org/documentation/latest/configuration/agent/)

launcher

The launcher to be used for Mesos containerizer. It could either be linux or posix

containerizers

Comma-separated list of containerizer implementations to compose in order to provide containerization. Available options are mesos and docker (on Linux).

isolation

isolation mechanisms to use, e.g., posix/cpu,posix/mem

imageProviders

Comma-separated list of supported image providers, e.g., APPC,DOCKER.

executorRegistrationTimeout

Defaults to one minute.

fetcherStallTimeout

Defaults to one minute and must be lower or equal to the executor registration timeout.
