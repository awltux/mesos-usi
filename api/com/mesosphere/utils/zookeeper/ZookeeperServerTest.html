ZookeeperServerTest 

trait ZookeeperServerTest extends BeforeAndAfterAll with StrictLogging

A trait that can be mixed in to run integration tests with a in-memory Zookeeper server. Note that because this trait uses BeforeAndAfterAll which invokes super.run to run the suite, you may need to mix this trait in last to get the desired behavior.
